# Qaos Visual Theme & Effects

## Visual Effects
*   **Glassmorphism:**
    *   Used in navigation bars, floating action menus, and the "Agent Status" overlay.
    *   **Effect:** Semi-transparency with a background blur (backdrop-filter).
    *   **Purpose:** Allows the user to see the "website under test" vaguely behind the control controls.

*   **The "Aura" System:**
    *   **Concept:** Instead of static backgrounds, use slowly moving, diffused color blobs ("Auras") to represent the AI's state.
    *   **Idle:** Soft Peach (`#FFEED8`) or Beige breathing effect.
    *   **Thinking (Strategist):** Violet (`#D0B2FF`) pulses.
    *   **Scanning (Visual Oracle):** Cyan (`#99FFF9`) sweeping gradients.
    *   **Action (Navigator):** Orange (`#E16540`) highlights.

*   **Shadows:**
    *   **Style:** Ultra-soft, large spread.
    *   **Usage:** To make the Qaos dashboard float "above" the application being tested.

## UI Elements
*   **Inputs:** `6px` rounded corners, clean borders. minimal styling.
*   **Icons:** Simple, distinct icons. Use the Agent Colors to color-code icons in the timeline (e.g., an eye icon in Green for Visual Oracle).
