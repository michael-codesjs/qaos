# Qaos Typography

## Font Family
*   **Primary Font:** `Labil Grotesk Variable`
    *   *Note:* If unavailable, use a high-quality geometric sans-serif like **Inter**, **Manrope**, or **DM Sans**.
*   **Code Monospace:** `JetBrains Mono` or `Fira Code`.
    *   *Usage:* Test logs, code snippets, selector identifiers.

## Heading Styles
*   **H1 / Hero:** Large, bold weights (up to `48px`+), tight letter spacing. Used for landing page statements like "Testing with Intent."
*   **H2 / Section Headers:** Bold, commanding. Used for dashboard sections ("Active Test Runs," "Chaos Scenarios").
*   **Color:** Primarily `#111111` (Near Black).

## Body Styles
*   **Base Size:** `13px` - `16px`.
*   **Weights:** Medium or Regular.
*   **Color:** `#5E5C5A` (Muted Grey).
*   **Readability:** essential for reading complex test logs and reasoning steps.

## UI Typography
*   **Labels & Badges:** Uppercase, tracked out (letter-spacing `0.05em`), small size (`11px` or `12px`).
*   **Navigation:** Precise, medium weight, clean hover states.
